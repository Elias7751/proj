# فحص نظام الإعدادات، الدعم، ولوحة الإدارة (Settings, Support & Admin)

## ملخص النظام
1. **الإعدادات:** نظام ديناميكي لحفظ إعدادات المنصة (مثل اسم المنصة، الشعار، معلومات التواصل) في قاعدة البيانات وجلبها بسهولة.
2. **الدعم الفني:** نظام تذاكر يتيح للعملاء وأصحاب المتاجر التواصل مع الإدارة لحل المشاكل، مع إمكانية إضافة أسئلة شائعة (FAQ).
3. **لوحة الإدارة:** مسارات مخصصة للمسؤولين (Admins) لإدارة المستخدمين، المتاجر، والطلبات بشكل مركزي.

## الأخطاء والمشاكل التي تم فحصها
1. **ارتباط الردود بالمستخدمين:**
   - تم إصلاح خطأ `Unknown column 'replies->user.name'` بتغييره إلى `fullName` ليتوافق مع نموذج المستخدم `User`.
2. **تحديث الإعدادات:**
   - يتم استخدام `upsert` في Sequelize لتحديث الإعدادات إذا كانت موجودة، أو إنشائها إذا لم تكن موجودة.
3. **حماية مسارات الإدارة:**
   - جميع مسارات `/api/v1/admin` محمية بـ `protect` و `authorize('admin')` لضمان عدم وصول غير المصرح لهم.
