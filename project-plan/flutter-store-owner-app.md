# خطة تطبيق التاجر (Store Owner App - Flutter)

## 📱 نظرة عامة
تطبيق التاجر هو الواجهة المخصصة لأصحاب المتاجر لإدارة متاجرهم، منتجاتهم، طلباتهم، والاشتراكات. سيتم بناء التطبيق باستخدام إطار عمل Flutter ليعمل على نظامي Android و iOS.

## 🛠️ التقنيات والمكتبات المستخدمة
- **إدارة الحالة:** `get` (GetX).
- **الشبكات:** `dio`.
- **التخزين المحلي:** `shared_preferences`.
- **الواجهة:** `google_fonts`، `flutter_svg`.
- **أخرى:** `image_picker` (لرفع صور المنتجات والمتجر)، `fl_chart` (للتحليلات والإحصائيات).

## 🗂️ هيكلية المجلدات
```
lib/
├── core/               # الأكواد المشتركة (الثوابت، الألوان، الثيم، الشبكة)
├── data/               # النماذج (Models)
├── features/           # الميزات الرئيسية
│   ├── auth/           # تسجيل الدخول، التسجيل كتاجر
│   ├── dashboard/      # لوحة التحكم الرئيسية (الإحصائيات)
│   ├── products/       # إدارة المنتجات (إضافة، تعديل، حذف)
│   ├── orders/         # إدارة الطلبات (تغيير الحالة)
│   ├── store_settings/ # إعدادات المتجر (الاسم، الشعار، أوقات العمل)
│   ├── subscriptions/  # الاشتراكات والباقات
│   └── offers_ads/     # إدارة العروض والكوبونات وطلب الإعلانات
├── routes/             # مسارات التنقل
└── main.dart           # نقطة البداية
```
