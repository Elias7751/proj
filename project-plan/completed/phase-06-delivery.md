## الـ API Endpoints
```
# المندوبين
POST   /api/v1/drivers/register            # تسجيل مندوب
GET    /api/v1/drivers/available            # مندوبين متاحين
PUT    /api/v1/drivers/status               # تحديث حالة المندوب
PUT    /api/v1/drivers/location             # تحديث الموقع

# التوصيل
POST   /api/v1/deliveries                   # تعيين توصيلة
GET    /api/v1/deliveries/:id               # تفاصيل التوصيلة
PUT    /api/v1/deliveries/:id/accept        # قبول التوصيلة
PUT    /api/v1/deliveries/:id/status        # تحديث حالة
GET    /api/v1/deliveries/my                # توصيلاتي (مندوب)
GET    /api/v1/delivery/calculate-fee       # حساب رسوم التوصيل
```
