## المهام

### 8.1 نموذج التقييم (Review Model)
- [x] إنشاء Review Schema (Sequelize Model)
  - المستخدم
  - المتجر / المنتج / المندوب
  - نوع التقييم (store, product, delivery)
  - عدد النجوم (1-5)
  - التعليق
  - الصور (اختياري)
  - الطلب المرتبط
  - حالة التقييم (active, hidden, reported)
  - رد صاحب المتجر

## الـ API Endpoints
```
POST   /api/v1/reviews                       # إضافة تقييم
GET    /api/v1/stores/:id/reviews            # تقييمات متجر
GET    /api/v1/products/:id/reviews          # تقييمات منتج
PUT    /api/v1/reviews/:id                   # تعديل تقييم
DELETE /api/v1/reviews/:id                   # حذف تقييم
POST   /api/v1/reviews/:id/reply             # رد المتجر
POST   /api/v1/reviews/:id/report            # تبليغ
```
