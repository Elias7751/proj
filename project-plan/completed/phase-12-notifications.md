## الـ API Endpoints
```
GET    /api/v1/notifications                 # إشعاراتي
GET    /api/v1/notifications/unread-count    # عدد غير المقروءة
PUT    /api/v1/notifications/:id/read        # تعيين كمقروء
PUT    /api/v1/notifications/read-all        # قراءة الكل
DELETE /api/v1/notifications/:id             # حذف إشعار
POST   /api/v1/notifications/send            # إرسال إشعار (Admin)
```
