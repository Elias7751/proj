## المهام

### 11.1 نموذج الباقة (Plan Model)
- [x] إنشاء Plan Schema (Sequelize Model)
  - اسم الباقة (مجاني، برونزي، فضي، ذهبي)
  - السعر الشهري
  - السعر السنوي (خصم)
  - المميزات:
    - عدد المنتجات المسموح
    - إعلانات مجانية
    - تقارير متقدمة
    - دومين فرعي مميز
    - تخصيص الألوان
    - أولوية في البحث
    - دعم فني مميز
    - نسبة عمولة أقل

### 11.2 تفاصيل الباقات
```
| الميزة                  | مجاني | برونزي | فضي  | ذهبي    |
|------------------------|-------|--------|------|---------|
| عدد المنتجات           | 20    | 500    | 2000 | غير محدود|
| العمولة                | 10%   | 7%     | 5%   | 3%      |
| تقارير متقدمة          | ❌    | ❌     | ✅   | ✅      |
| إعلانات مجانية         | 0     | 1      | 3    | 5       |
| تخصيص الألوان          | ❌    | ❌     | ✅   | ✅      |
| رابط مميز              | ❌    | ✅     | ✅   | ✅      |
| دعم فني مميز           | ❌    | ❌     | ✅   | ✅      |
| أولوية في البحث        | ❌    | ❌     | ❌   | ✅      |
```

## الـ API Endpoints
```
# الباقات
GET    /api/v1/plans                         # كل الباقات
GET    /api/v1/plans/:id                     # تفاصيل باقة

# الاشتراكات
POST   /api/v1/subscriptions                 # اشتراك جديد
GET    /api/v1/subscriptions/my              # اشتراكي الحالي
PUT    /api/v1/subscriptions/upgrade         # ترقية
PUT    /api/v1/subscriptions/cancel          # إلغاء
GET    /api/v1/subscriptions/history         # سجل الاشتراكات
```
