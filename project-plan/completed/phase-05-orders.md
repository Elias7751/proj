## الـ API Endpoints
```
# السلة
POST   /api/v1/cart                        # إضافة للسلة
GET    /api/v1/cart                         # عرض السلة
PUT    /api/v1/cart/:itemId                # تعديل الكمية
DELETE /api/v1/cart/:itemId                # حذف من السلة
DELETE /api/v1/cart                         # تفريغ السلة

# الطلبات
POST   /api/v1/orders                      # إنشاء طلب
GET    /api/v1/orders/my                   # طلباتي
GET    /api/v1/orders/:id                  # تفاصيل طلب
PUT    /api/v1/orders/:id/status           # تحديث حالة (متجر)
PUT    /api/v1/orders/:id/cancel           # إلغاء طلب
GET    /api/v1/stores/:id/orders           # طلبات المتجر
```
