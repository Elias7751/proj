## المهام

### 3.1 نموذج المتجر (Store Model)
- [x] إنشاء Store Schema (Sequelize Model)
  - اسم المتجر (عربي + إنجليزي)
  - الشعار (logo)
  - صورة الغلاف (cover)
  - الرابط المخصص (slug): `platform.com/store-name`
  - الوصف
  - نوع المتجر (مطعم، بقالة، صيدلية، ملابس...)
  - رقم واتساب
  - الموقع على الخريطة (lat, lng)
  - العنوان (المدينة، المنطقة، الشارع)
  - ساعات العمل
  - سياسة التوصيل
  - الحد الأدنى للطلب
  - التقييم (متوسط)
  - حالة المتجر (pending, active, blocked, closed)
  - صاحب المتجر (مرجع User)
  - باقة الاشتراك (مرجع Subscription - لاحقاً)

### 3.2 نموذج التصنيفات (Category Model)
- [x] تصنيفات رئيسية (مطاعم، ملابس، إلكترونيات...)
- [x] تصنيفات فرعية (علاقة ذاتية)
- [x] أيقونة/صورة لكل تصنيف
- [x] ترتيب العرض

### 3.3 نموذج المدن والمناطق
- [x] City Schema (صنعاء، عدن، تعز، إب، الحديدة...)
- [x] Area Schema (مناطق داخل كل مدينة)
- [x] حالة التغطية (مفعل/معطل)

### 3.4 عمليات المتاجر
- [x] إنشاء متجر جديد (طلب انضمام)
- [ ] الموافقة/رفض المتجر (Admin)
- [x] تعديل بيانات المتجر
- [x] عرض المتجر (صفحة المتجر المخصصة عبر slug)
- [x] البحث عن المتاجر + فلترة
- [ ] المتاجر القريبة (حسب الموقع)
- [x] المتاجر حسب التصنيف
- [x] المتاجر حسب المدينة

### 3.5 إدارة موظفي المتجر
- [ ] إضافة موظف للمتجر

- [ ] إزالة موظف

## الـ API Endpoints
```
# المتاجر
POST   /api/v1/stores                    # إنشاء متجر
GET    /api/v1/stores                     # كل المتاجر
GET    /api/v1/stores/:slug              # متجر بالرابط
GET    /api/v1/stores/nearby             # متاجر قريبة
PUT    /api/v1/stores/:id                # تعديل متجر
DELETE /api/v1/stores/:id                # حذف متجر

# التصنيفات
POST   /api/v1/categories                # إنشاء تصنيف
GET    /api/v1/categories                 # كل التصنيفات
GET    /api/v1/categories/:id/stores     # متاجر تصنيف معين
PUT    /api/v1/categories/:id            # تعديل
DELETE /api/v1/categories/:id            # حذف

# المدن والمناطق
GET    /api/v1/cities                     # كل المدن
GET    /api/v1/cities/:id/areas          # مناطق مدينة
POST   /api/v1/cities                    # إضافة مدينة (Admin)
POST   /api/v1/cities/:id/areas          # إضافة منطقة (Admin)
```
