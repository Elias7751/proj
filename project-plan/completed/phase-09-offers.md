## الـ API Endpoints
```
# العروض
POST   /api/v1/offers                        # إنشاء عرض
GET    /api/v1/offers                         # عروض نشطة
GET    /api/v1/stores/:id/offers             # عروض متجر
PUT    /api/v1/offers/:id                    # تعديل عرض
DELETE /api/v1/offers/:id                    # حذف عرض

# الكوبونات
POST   /api/v1/coupons                       # إنشاء كوبون
POST   /api/v1/coupons/validate              # التحقق من كوبون
GET    /api/v1/coupons                        # كل الكوبونات (متجر)
PUT    /api/v1/coupons/:id                   # تعديل
DELETE /api/v1/coupons/:id                   # حذف
```
