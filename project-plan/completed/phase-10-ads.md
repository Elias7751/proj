## المهام

### 10.1 نموذج الإعلان (Ad Model)
- [x] إنشاء Ad Schema (Sequelize Model)
  - عنوان الإعلان
  - الصورة/البانر
  - رابط الإعلان
  - المتجر المعلن
  - موضع الإعلان:
    - الصفحة الرئيسية (بانر)
    - أعلى نتائج البحث
    - داخل تصنيف معين
    - بين المنتجات
  - تاريخ البداية
  - تاريخ الانتهاء
  - التكلفة
  - عدد المشاهدات
  - عدد النقرات
  - الحالة (pending, active, expired, rejected)

- [x] صاحب المتجر يطلب إعلان
- [x] اختيار الموضع والمدة
- [x] حساب التكلفة
- [ ] الدفع
- [x] الموافقة/الرفض (Admin)

### 10.3 عرض الإعلانات
- [x] عرض الإعلانات حسب الموضع
- [x] تتبع المشاهدات والنقرات
- [ ] Rotation للإعلانات (تبديل)

### 10.4 تقارير الإعلانات

- [ ] CTR (نسبة النقر)

## الـ API Endpoints
```
POST   /api/v1/ads                           # طلب إعلان
GET    /api/v1/ads/placements/:position      # إعلانات بموضع
GET    /api/v1/ads/my                        # إعلاناتي
GET    /api/v1/ads/:id/stats                 # إحصائيات إعلان
PUT    /api/v1/ads/:id                       # تعديل
PUT    /api/v1/ads/:id/approve               # موافقة (Admin)
POST   /api/v1/ads/:id/click                 # تسجيل نقرة
```
