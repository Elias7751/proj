## الـ API Endpoints
```
POST   /api/v1/stores/:storeId/products        # إضافة منتج
GET    /api/v1/stores/:storeId/products        # منتجات متجر
GET    /api/v1/products                         # كل المنتجات (بحث + فلترة)
GET    /api/v1/products/:id                    # تفاصيل منتج
PUT    /api/v1/products/:id                    # تعديل منتج
DELETE /api/v1/products/:id                    # حذف منتج
PUT    /api/v1/products/:id/toggle             # إخفاء/إظهار
POST   /api/v1/products/:id/images             # رفع صور
DELETE /api/v1/products/:id/images/:imageId    # حذف صورة
GET    /api/v1/products/featured               # منتجات مميزة
GET    /api/v1/products/best-selling           # الأكثر مبيعاً
```
