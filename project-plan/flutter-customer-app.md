# خطة تطبيق العميل (Customer App - Flutter)

## 📱 نظرة عامة
تطبيق العميل هو الواجهة الرئيسية للمستخدمين لتصفح المتاجر، المنتجات، وإتمام عمليات الشراء. سيتم بناء التطبيق باستخدام إطار عمل Flutter ليعمل على نظامي Android و iOS.

## 🛠️ التقنيات والمكتبات المستخدمة (Packages)
- **إدارة الحالة (State Management):** `get` أو `provider` أو `riverpod` (سنستخدم `get` لسهولته وسرعته في التطوير).
- **الشبكات (Networking):** `dio` أو `http` للاتصال بـ Backend API.
- **التخزين المحلي (Local Storage):** `shared_preferences` أو `get_storage` لحفظ التوكن (Token) وبيانات المستخدم.
- **الواجهة (UI):** `google_fonts`، `cached_network_image`، `flutter_svg`.
- **أخرى:** `url_launcher` (لفتح روابط واتساب)، `geolocator` (للموقع).

## 🗂️ هيكلية المجلدات (Folder Structure)
سنعتمد هيكلية تعتمد على الميزات (Feature-first) داخل مجلد `lib`:
```
lib/
├── core/               # الأكواد المشتركة (الثوابت، الألوان، الثيم، الشبكة)
├── data/               # النماذج (Models) ومزودي البيانات (Providers/Services)
├── features/           # الميزات الرئيسية للتطبيق
│   ├── auth/           # تسجيل الدخول، التسجيل
│   ├── home/           # الصفحة الرئيسية (التصنيفات، العروض)
│   ├── stores/         # قائمة المتاجر وتفاصيل المتجر
│   ├── products/       # تفاصيل المنتج
│   ├── cart/           # سلة المشتريات وإتمام الطلب
│   ├── orders/         # طلباتي وتتبع الطلب
│   └── profile/        # الملف الشخصي والإعدادات
├── routes/             # مسارات التنقل (Navigation)
└── main.dart           # نقطة البداية
```
