# توثيق واجهات برمجة التطبيقات (API Documentation)
هذا الملف يحتوي على توثيق شامل لجميع مسارات الـ API الخاصة بمنصة "السوق اليمني" مقسمة حسب نوع المستخدم (عميل، تاجر، أدمن).

الرابط الأساسي (Base URL): `http://localhost:5000/api/v1`

---

## 1. المصادقة (Authentication) - عام
| المسار (Endpoint) | الطريقة (Method) | الوصف | الصلاحية |
| :--- | :--- | :--- | :--- |
| `/auth/register` | POST | تسجيل حساب جديد (عميل/تاجر) | عام |
| `/auth/login` | POST | تسجيل الدخول | عام |
| `/auth/me` | GET | جلب بيانات المستخدم الحالي | مسجل الدخول |

---

## 2. تطبيق العميل (Customer App)

### المتاجر والمنتجات
| المسار (Endpoint) | الطريقة (Method) | الوصف | الصلاحية |
| :--- | :--- | :--- | :--- |
| `/stores` | GET | جلب جميع المتاجر النشطة | عام |
| `/stores/:id` | GET | جلب تفاصيل متجر معين | عام |
| `/products` | GET | جلب المنتجات (مع دعم البحث والفلترة) | عام |
| `/products/:id` | GET | جلب تفاصيل منتج معين (يزيد عدد المشاهدات) | عام |
| `/categories` | GET | جلب جميع التصنيفات | عام |

### المفضلات (Favorites)
| المسار (Endpoint) | الطريقة (Method) | الوصف | الصلاحية |
| :--- | :--- | :--- | :--- |
| `/favorites` | GET | جلب قائمة مفضلات العميل | عميل |
| `/favorites/toggle` | POST | إضافة/إزالة منتج من المفضلة | عميل |
| `/favorites/check/:productId` | GET | التحقق مما إذا كان المنتج في المفضلة | عميل |

### الطلبات ونظام التواصل (Lead System)
| المسار (Endpoint) | الطريقة (Method) | الوصف | الصلاحية |
| :--- | :--- | :--- | :--- |
| `/orders/lead` | POST | تسجيل طلب تواصل جديد (Lead) من صفحة المنتج | عميل |
| `/orders/my` | GET | جلب سجل طلبات التواصل الخاصة بالعميل | عميل |
| `/orders/:id` | GET | جلب تفاصيل طلب معين | عميل / تاجر |

---

## 3. تطبيق التاجر (Store Owner App)

### إدارة المتجر
| المسار (Endpoint) | الطريقة (Method) | الوصف | الصلاحية |
| :--- | :--- | :--- | :--- |
| `/stores/register` | POST | تسجيل متجر جديد (يتطلب موافقة الإدارة) | تاجر |
| `/stores/my-store` | GET | جلب بيانات متجر التاجر الحالي | تاجر |
| `/stores/my-store` | PUT | تحديث بيانات المتجر (الاسم، الوصف، الواتساب) | تاجر |

### إدارة المنتجات
| المسار (Endpoint) | الطريقة (Method) | الوصف | الصلاحية |
| :--- | :--- | :--- | :--- |
| `/products/store/my` | GET | جلب جميع منتجات التاجر | تاجر |
| `/stores/:storeId/products` | POST | إضافة منتج جديد | تاجر |
| `/products/:id` | PUT | تعديل بيانات منتج | تاجر |
| `/products/:id` | DELETE | حذف منتج | تاجر |

### إدارة الطلبات (Leads)
| المسار (Endpoint) | الطريقة (Method) | الوصف | الصلاحية |
| :--- | :--- | :--- | :--- |
| `/orders/store` | GET | جلب جميع طلبات التواصل الواردة للمتجر | تاجر |
| `/orders/:id/status` | PUT | تحديث حالة الطلب (تواصل، قيد التنفيذ، مكتمل) | تاجر / أدمن |

### الاشتراكات والباقات
| المسار (Endpoint) | الطريقة (Method) | الوصف | الصلاحية |
| :--- | :--- | :--- | :--- |
| `/subscriptions/plans` | GET | جلب الباقات المتاحة للاشتراك | تاجر / أدمن |
| `/subscriptions/my` | GET | جلب تفاصيل اشتراك التاجر الحالي | تاجر |
| `/subscriptions/subscribe` | POST | طلب الاشتراك في باقة معينة | تاجر |
